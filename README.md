# -
Vue2.0+Node.js+MongoDB+axios

实现了的功能有：
前台
 1.  登录
 2.  商品展示
 3.  购买商品
 4.  购物车
 5.  购物车全选，单选
 5.  购物车商品删除 编辑数量(页面刷新,编辑数量不变)
 6.  购物地址,删除/添加/编辑购物地址
 7.  模拟手机验证码付款
 8.  付款成功跳转成功页面
 
后台 
 1.  管理员用户登录
 2.  登陆后用户信息显示在头部
 3.  实现当月畅销
 4.  实现美妆套装
 5.  官方产品  
 6.  官方产品上传
 7.  读取购买的订单
